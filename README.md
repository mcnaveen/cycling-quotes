## Installation 📦

- Clone this repo or Download

```sh
git clone https://github.com/mskian/cycling-quotes
cycling-quotes
yarn install
```

- Test the site

```sh
yarn dev
```

- Build the site

```sh
yarn build
```

- Test the production Build on Localhost

```sh
yarn start
```
